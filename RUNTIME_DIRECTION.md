# Runtime Direction

.NeoN should remain implementation-neutral.

Python is the first reference implementation.

Future implementations may include:

- Mojo
- Rust
- Go
- browser runtimes
- local verification engines

## Long-Term Direction

The system evolves toward lineage-aware computation.

Meaning:

Data carries continuity metadata during transformation.

## Near-Term Focus

Do not begin with browser surgery.

Build:

- stable object model
- deterministic hashing
- lineage graph semantics
- content-addressable storage
- portable proof packets

first.
