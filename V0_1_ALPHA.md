# .NeoN v0.1 Alpha

Goal:
Create the first stable WorkLedger reference implementation.

Success criteria:

- initialize a local vault
- create native .neon artifacts
- validate artifact structure
- export proof packets
- verify hashes
- track lineage relationships
- remain local-first and dependency-light

Non-goals:

- token systems
- cloud dependency
- enterprise SaaS
- social features
- marketplaces

The first release proves the primitive, not the ecosystem.
