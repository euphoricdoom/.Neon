# Example Chain

This folder shows a simple parent-child artifact chain for .NeoN.

Order:

1. creator.neon
2. invoice-workflow.neon
3. ai-assisted-invoice-workflow.neon
4. symbolic-compressed-invoice-workflow.neon
5. proof-enhanced-invoice-workflow.neon

Goal:

Show that each artifact can preserve a parent reference and build an inspectable chain.
