# Example Artifacts

This folder contains small, versioned, testable example artifacts for .NeoN.

## Rule

Examples must be:

- Small and focused
- Versioned
- Testable against the current schema and CLI
- Representative of real usage patterns

Do not add decorative or aspirational examples. Examples exist to enforce reference behavior.
