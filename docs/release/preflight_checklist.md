# Workflow Preflight Checklist

Use this checklist before starting a major repo/agent pass.

## Preflight Questions

- [ ] Is there already an open PR for this work?
- [ ] Is the current branch state reviewed?
- [ ] Is the workbook task created (ID/link ready)?
- [ ] Is there a GitHub issue for this work?
- [ ] Is the release target known?
- [ ] Does this create or change markdown? If yes, what category?
- [ ] Does this duplicate Drive/workbook/source-of-truth content?
- [ ] Is CI expected to run for this change?
- [ ] What evidence row will be updated after completion?

## Post-Merge Requirement

- [ ] Workbook row/task update is required after merge and has an owner.
