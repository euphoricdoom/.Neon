# Archive

This folder holds obsolete planning markdowns that have been superseded but not yet deleted.

## Rule

A file belongs here only when:

1. Its content has been mapped to the workbook, Drive, or a stable repo location
2. It is no longer the active source of truth for anything
3. Deleting it would lose no unique protocol history

Do not delete old docs without first confirming their content is captured elsewhere.
Move here as a staging step before permanent deletion.
