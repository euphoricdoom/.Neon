# Canonical Spec Flow

This folder contains stable, exported specs from the NeoN governance model.

## Rule

A spec belongs here only when it is:

1. Defined and stable in the workbook or Drive
2. Versioned (e.g., `_v0.1`)
3. Ready to be referenced by tests or validators

Do not paste draft or in-progress specs here. Keep those in Drive until stable.
